### 3장. 자바스크립트에서 비동기 처리 다루기      
   
✔ Promise   
   
    ◼ 비동기 작업을 조금 더 편하게 처리할 수 있도록 하는 역할
    ◼ Promise는 성공할 수도 있고 실패할 수도 있음    
    ◼ Promise 생성   
        ◾ const myPromise = new Promise((resolve, reject) => { ... })   
        ◾ 성공할 때는 resolve 호출, 실패할 때는 reject 호출
